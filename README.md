<h1 align="left">微信小程序开发助手</h1>

<img src="https://github.com/overtrue/vscode-miniapp-helper/blob/master/assets/images/logo.png" height="16"/> 微信小程序开发助手 for VSCode，代码提示 + 语法高亮

## Installation

- 打开 VSCode
- 拓展中心搜索 "小程序开发助手"
- 点击安装

## Features

- 支持 js，json，wxml 文件的代码提示
- wxml 文件语法高亮
